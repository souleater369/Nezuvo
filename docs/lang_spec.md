# Nezuvo Language Specification

## Purpose

Nezuvo is a beginner-friendly scripting language designed for learning programming concepts, compiler construction, and AI scripting.


## Variables

Variables are created using the nez keyword:

nez age = 18


## Data Types

Nezuvo supports:

- Integer
- Float
- String
- Boolean
